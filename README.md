# Accident Severity Machine Learning Analysis

This project applies machine learning classification models to predict road accident severity using environmental and road-related factors.

## Features used
- Weather Conditions
- Light Conditions
- Junction Detail
- Speed Limit

## Models applied
- Logistic Regression
- Decision Tree
- Random Forest

The Random Forest model provided the most meaningful results for identifying fatal accidents and highlighted speed limit as the strongest predictor of accident severity.

This project supports transport safety policy recommendations by identifying key risk factors associated with severe road accidents.
